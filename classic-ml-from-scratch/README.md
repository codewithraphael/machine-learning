# Classic Machine Learning From Scratch

A hands-on repository for implementing classic machine learning algorithms from scratch using Python. The goal is to build a strong understanding of the core ideas behind each model by coding them directly, without relying on high-level machine learning libraries.

## What this repository includes

This project focuses on foundational algorithms such as:
- Linear Regression
- Logistic Regression
- K-Nearest Neighbors
- Naive Bayes
- Decision Trees
- K-Means Clustering

Each implementation is designed to be simple, educational, and easy to follow.

## Why this project exists

The purpose of this repository is to help learners understand:
- how machine learning algorithms work internally
- the mathematics behind common models
- the training and prediction process
- how to evaluate and compare results

## Getting started

1. Clone the repository.
2. Install the required dependencies.
3. Explore the algorithm implementations in the `algorithms` folder.
4. Run the examples and experiment with your own data.

## Example

You can start by testing a simple linear regression or k-means clustering example using the provided scripts or notebooks.
