# Predictive Maintenance Dataset

This dataset contains engine sensor parameters such as RPM, lubrication oil pressure, fuel pressure, coolant pressure, lubrication oil temperature, and coolant temperature used for predictive maintenance and engine condition classification.

The dataset will be used for developing machine learning models capable of identifying whether an engine requires maintenance or is operating normally.

## Features
- Engine RPM
- Lubrication Oil Pressure
- Fuel Pressure
- Coolant Pressure
- Lubrication Oil Temperature
- Coolant Temperature
- Engine Condition (Target)

## Intended Use
This dataset is intended for educational and predictive maintenance machine learning applications.
